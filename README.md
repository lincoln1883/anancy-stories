# 📖 Anancy Stories<a name="about-project"></a>

The Anancy Stories app is a simple bookstore where you can add and delete books. features include Typescript, React Hooks,React Router, State Management with Redux & Redux Toolkit and API calls with Axios.

### Prerequisites

In order to run this project you need to have the following installed:

```sh
- A web browser
- A code editor
- A terminal
- Git
- Node.js
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder run `git clone https://github.com/lincoln1883/anancy-stories/`
```

### Install

Install this project with:

```sh
  cd my-folder run `npm install`
```

### Usage

To run the project, execute the following command:
```sh
npm run dev
```
